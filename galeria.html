Esta galeria esta contruida con transformaciones de css3, se usaron especificamente las trasformaciones "scale" "rotate" y "translate". Tambien se uso opacity y tansiciones.

Cuando se teclea sobre una imagen se uso un lightbox de jquery para agrandar las imagenes.
